A case for Raspberry Pi 5 + PiEEG-16 to be mounted to the current IronBCI 3D-printable headset.

Remix of: https://www.printables.com/model/742926-raspberry-pi-5-case/files

Used on: https://www.instructables.com/Open-Source-EEG-3D-Printable-Headset-for-Brain-Com/


----

Print Instructions

Case body: print bottom-side down, limited if any support needed.

Mountable case lid: print wing-side down (latch-side up) with standard support, 30% gyroid infill, 0.12mm layer height.

----

Build Instructions

1) Attach RPi and PiEEG to case with standard standoffs and mounting hardware.

2) Mount the lid to the headset with M2.5x12 screws and nuts. 

3) If necessary for additional stability, apply velcro belt around the unit. 

See pictures in /images folder for further guidance.
